# YoctoRpi5
Yocto image for my Raspberry 5

## Current features
* Console outputs to GPIO serial UART pins (ttyAMA0)
* Systemd starts ssh service at startup
* Systemd starts ifupdown service at startup for eth0 with static ip of 192.168.2.2
